## Setup to create build

1. Make sure you are using python3
`which python`
2. Install `poetry`
```curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python```

## Run Setup project
1. `poetry install`


If adding library
2. `poetry install pandas`
